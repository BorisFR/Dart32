Choisissez votre jeu

Débutant

Le COUNT-UP est parfait pour s'entraîner seul. Le jeu est simple, c'est simplement l'addition des points des 24 lancers.

En partant de 0, les joueurs marquent et accumulent des points. Trois lancers par round et les joueurs essaient d'obtenir le plus de points en 8 rounds.

Partis de 0 points, le joueur avec le plus de points gagne.

Match sur 8 rounds, 3 lancers par round.
Classique

Très simple! Le 01 est un des jeux standard aux fléchettes. Ce jeu est joué dans les principales compétitions.

Il y a 6 types de 01: 301, 501, 701, 901, 1101 et 1501.

Le jeu commence avec un nombre donné de points. Si le jeu est un 301, tous les joueurs commencent avec 301 points. Le nom du jeu est le nombre de points de départ. Le nombre touché est soustrait aux points de départs, et le jeu se termine quand le score arrive à 0 exactement ! Le premier joueur à réduire son score à 0 gagne !

Quand vos points descendent en dessous de zéro, c'est un BUST. Dans le cas d'un BUST, votre tour s'arrête, même si vous n'avez pas lancé vos 3 flèches. Votre prochain tour recommencera au score que vous aviez avant le BUST.
Habitué

Essayez le CRICKET dès que vous vous êtes habitué aux fléchettes.

Il faut connaître quelques règles pour ce jeu stratégique, mais une fois apprise, c'est un jeu très prenant.

Les nombres utilisés sont 15, 16, 17, 18, 19, 20 et la BULL. Les autres nombres ne rapportent pas de points.

Quand vous marquez 3 marques dans un nombre, celui-ci est ouvert. Atteindre ce nombre à partir de ce moment vous rapportera des points.

Si votre adversaire met 3 marques dans ce nombre, il est fermé. La zone devient "invalide". C'est une "fermeture".

Le joueur qui a le plus de points lorsque tous les nombres sont fermés, ou lorsque le nombre de round limite est atteint, gagne.
